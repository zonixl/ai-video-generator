## 任务
为 image 模板生成画面描述和标题。不需要 components，系统会自动生成 AI 插画。

## 可用模板（根据分镜调性选一个）
- image_full：全屏背景——大气/史诗/自然/纪录片
- image_elegant：雅致插画——人文/哲理/情感/成长
- image_card：信息卡片——科普/知识/教育
- image_modern：现代科技——科技/商业/趋势
- image_neon：霓虹赛博——潮流/游戏/音乐/年轻化

## 规则
1. visual 用英文，适合文生图模型，描述具体画面场景，不要写文字/字幕/logo/水印
2. image_full 模板的 visual 必须包含"画面上方和底部偏暗，适合白色文字叠放"的构图要求
3. headline 是本镜专属要点，5-15 字
4. tts_emotion 选一个：自信坚定 / 温柔亲切 / 娓娓道来 / 严肃沉稳 / 兴奋激动 / 急促紧张 / 低沉神秘 / 叙事平缓

## 输出 JSON 格式
{{
  "scene_index": 1,
  "duration": 5.0,
  "template": "image_elegant",
  "theme": "warm_grid",
  "headline": "本镜专属要点",
  "subtitle": "字幕文本",
  "visual": "English description for image generation",
  "tts_emotion": "温柔亲切",
  "components": []
}}

## 分镜信息
- 序号：{index}  时长：{duration} 秒
- 字幕：{subtitle}
